Class EaseShared

Všeobecně sdílený objekt frameworku. Tento objekt je automaticky přez svůj singleton instancován do každého Ease* objektu. Poskytuje kdykoliv přístup k často volaným objektům framworku jako například uživatel, databáze, webstránka nebo logy. Také obsahuje pole obecnych nastavení a funkce pro jeho obluhu.
